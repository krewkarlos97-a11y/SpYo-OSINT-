# Program and Web App Databases

## Program and Web App Databases

Use this page to quickly find databases, directories, and discovery platforms for software tools.

### Program databases

* [AlternativeTo](https://alternativeto.net/) — Find desktop and mobile software alternatives.
* [SourceForge](https://sourceforge.net/software/) — Large software directory with categories and filters.
* [FOSSHub](https://www.fosshub.com/) — Catalog of free and open-source desktop software.
* [PortableApps](https://portableapps.com/apps) — Portable Windows applications database.
* [Chocolatey Community Packages](https://community.chocolatey.org/packages) — Windows package index for installable software.
* [GitHub Topics](https://github.com/topics) — Browse software projects by topic and tag.

### Web app databases

* [BuiltWith](https://builtwith.com/) — Detect technologies used by websites.
* [Wappalyzer](https://www.wappalyzer.com/) — Identify web stacks, frameworks, and services.
* [StackShare](https://stackshare.io/tools) — Explore tools used in modern software stacks.
* [Product Hunt](https://www.producthunt.com/) — Discover new web apps and SaaS products.
* [G2](https://www.g2.com/) — Business software marketplace and review database.
* [Capterra](https://www.capterra.com/) — Search business software by category and use case.

{% hint style="info" %}
Use program databases to find installable software. Use web app databases to research SaaS tools and site technology stacks.
{% endhint %}

### Suggested workflow

1. Start with a broad category.
2. Filter by platform, license, or use case.
3. Compare alternatives across multiple databases.
4. Validate current usage with web stack detection tools.
